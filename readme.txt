== Description ==
Using this plugin we can easily add notifications/messages at the top / bottom / after content section of the post.It uses marquee ,so it this works perfect in HTML5 themes

== Installation ==

1. Upload 'meassagetrigger' folder  to the '/wp-content/plugins/'' directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Go to Settings->Message Trigger and set the messages
4. Go to specific post you will separate box for trigger message
